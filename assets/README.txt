This folder stores the portfolio's local media files and generated PWA assets.

Current files used by the site:

- `YDMW-303.JPEG`: hero headshot image.
- `marcus_williams_resume.pdf`: downloadable resume.
- `icons/`: PWA install icons used by `manifest.json` and `index.html`.
- `splash/`: PWA launch image used by iPhone startup metadata in `index.html`.

Notes:

- If you rename the headshot or resume, update the references in `index.html`.
- If you rename PWA icons or splash images, update `manifest.json`, `index.html`, and `service-worker.js`.
- The icon and splash files are placeholders and can be replaced with branded assets later.
